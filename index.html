History

  • 2004 Safari browser

  • 2014 standardized by WHATWG

Canvas features

  • rectangles
  • polygon shapes
  • circles and arcs
  • Bezier curves
  • fill methods
  • text
  • images
  • videos
  • transformation
  • rotating
  • scaling
  • animation
  • pixel manipulation

Usage

  • animation
  • drawing graphics
  • 2D games
  • data visualization
  • photo-manipulation
  • real-time video processing
